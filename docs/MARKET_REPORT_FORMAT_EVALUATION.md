# 시장조사 결과 리포트 양식 평가 및 개선안

## 1. 현재 양식 평가

### 포함 항목 (강점)
- **제목·생성일·조직·국가**: 명확한 메타정보
- **의뢰 내역**: 아이템, 국가, HS 코드
- **시장 장악(market_dominance)**: 조사유형별 상위 5개 파트너국/업체, 점유율·근거 소스
- **관련 업체(related_companies)**: 5개, 연락처 출처·추천 이유
- **뉴스 요약**: 최대 10~12건, 제목·요약·출처·B2B/B2C 구분
- **다중 포맷**: PDF, DOCX, Excel, JSON 지원

### 약점 및 리스크
| 항목 | 내용 |
|------|------|
| **언어** | PDF/DOCX 본문이 영어 고정 구간 있음 (예: "Market dominance", "Related companies"). 옵션의 `language`가 리포트 본문에 완전 반영되지 않음. |
| **구조** | DOCX에서 market_dominance 루프가 제목 "Market dominance"를 유형마다 반복 출력할 수 있음. |
| **데이터 부족 시** | HS 미입력 시 전품목(TOTAL)만 표시되며, "이 결과는 전품목 기준입니다" 같은 안내 문구 없음. |
| **뉴스** | 국가별 뉴스 키 미설정 시 RSS만 사용된다는 점이 리포트에 표시되지 않음. |
| **1페이지 업체 요약** | 현재 없음. 프레임워크상 "업체당 1페이지 웹 요약"은 미구현. |
| **가격·진입 가능성** | 별도 섹션 없음. |

## 2. 개선안

1. **다국어 일관화**  
   리포트 제목·섹션명·라벨을 `ReportData.lang` 기준으로 완전히 다국어화 (PDF/DOCX/Excel 공통). 이미 `TITLES`, `reportSectionOptions` 등에 언어별 텍스트가 있으면 해당 값을 본문에 사용.

2. **DOCX 구조 정리**  
   "Market dominance" 제목은 한 번만 출력하고, 그 아래 각 조사유형별로 소제목 + 테이블만 반복하도록 수정.

3. **데이터 출처·범위 안내 문구**  
   - HS 코드가 있으면 "HS {code} 기준 수출·수입" 문구 추가.  
   - 없으면 "전품목(TOTAL) 기준" 문구 추가.  
   - 뉴스가 RSS만 사용된 경우 "(뉴스: RSS 공통 헤드라인)" 등 짧은 푸터/메모 추가.

4. **업체 1페이지 요약** ✅ 반영됨  
   - **주요 바이어/업체 Top 10**: 시장 장악(수입/수출 파트너국) 및 관련 업체 리스트를 각 10건까지 표기.  
   - **1~3위 업체 상세**: 관련 업체 1~3위에 대해 OpenCorporates로 회사 정보 검색 후, 1업체당 1페이지 분량으로 요약.  
   - 포함 항목: 업체명·국가·품목/HS·업체 유형·연락처(URL·사업자번호·설립일)·해당국 수입/수출 규모 안내·제품 선호/지역 문구·추천 이유.  
   - PDF/DOCX/Excel/JSON 모두 Top 10 및 Top 3 상세 섹션 포함.

5. **가격·진입 가능성 섹션 (중기)**  
   World Bank·IMF 등 지표를 "시장 규모·진입 난이도" 등으로 요약하는 문단 또는 표 추가. 전용 에이전트 확장 시 해당 출력을 리포트에 병합.

6. **에러/부분 실패 시**  
   세분화 분석 또는 뉴스 중 일부만 실패해도 나머지로 리포트 생성하고, "일부 데이터를 불러오지 못했습니다" 같은 경고 문구를 리포트 상단 또는 해당 섹션에 표시.

위 개선안 반영 순서 제안: 1(다국어) → 2(DOCX 구조) → 3(출처·범위 안내) → 4·5·6은 로드맵에 따라 단계적 적용.
