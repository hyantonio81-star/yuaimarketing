# 시장 조사 프레임워크 (Market Research Framework)

시장 인텔 리포트 및 에이전트 팀이 채우는 항목을 하나의 프레임워크로 정리한 문서입니다.

## 1. 적용 범위

| 영역 | 내용 | 현재 연동 |
|------|------|-----------|
| **제품·선호** | 소셜/커뮤니티 구매·공유 신호, 제품별 선호도 | 소셜(YouTube/Facebook API 연동 가능), 리뷰 툴 정의됨 |
| **주요 미디어·채널** | 국가별 주요 매체, B2B 채널 | 뉴스 요약(NewsData/Mediastack/RSS), SOURCES에 매체 목록 |
| **무역 API (HS)** | HS 코드별 수출·수입, 수출입 상위국·금액 | **UN Comtrade** (의뢰 `hs_code` 4/6자리 반영) |
| **수출입업체·1페이지 요약** | 수출입 상위 업체명, 업체당 1페이지 웹 요약 | **OpenCorporates** 기반 관련 업체 추천, 1페이지 요약은 추후(검색/스크래핑) |
| **정부 HS·기업 데이터** | 관세청/통계청 등 정부 HS·기업 공개 데이터 | DGCP(도미니카 조달), 기타 정부 API는 SOURCES 확장 대상 |
| **주요 기업·시장점유율** | 상위 업체, 점유율(거래액 기준) | **Comtrade** 파트너국별 비중 → market_dominance.top_players |
| **시장 가격·진입 가능성** | 가격대, 진입 장벽·가능성 | 지표(World Bank GDP 등) 보강 중, 전용 가격/진입 에이전트 확장 가능 |

## 2. 에이전트별 담당

- **TradeAgent**: HS 코드별 수출/수입, 상위 파트너국(시장점유율 근거).
- **CompanyAgent**: 국가·키워드 기준 기업 검색(관련 업체 추천).
- **NewsAgent**: 국가별 뉴스 요약(주요 미디어·채널 신호).
- **Supervisor**: 위 에이전트 결과를 한 번에 수집해 리포트용 데이터로 조합.

추가 예정: 정부 데이터 에이전트(DGCP 등), 소셜/선호 에이전트, 가격·진입 가능성 전용 에이전트.

## 3. 리포트 파이프라인과의 대응

- **세분화 분석 결과**: 무역(HS)·주요 기업·점유율 → `segmentedResult` (Comtrade + World Bank + OpenCorporates).
- **뉴스 요약**: 주요 미디어·채널 → `newsItems`.
- **리포트 생성**: `buildReportData()` → Supervisor → PDF/DOCX/Excel/JSON 출력.

이 프레임워크는 제품 선호, 주요 미디어, 무역(HS), 기업·점유율, 정부 데이터, 가격·진입 가능성까지 단계적으로 커버하도록 설계되어 있으며, 현재는 무역(HS)·기업·뉴스·경제지표까지 반영되어 있다.
