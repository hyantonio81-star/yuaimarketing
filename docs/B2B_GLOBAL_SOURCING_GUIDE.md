# yuaimarketing 글로벌 소싱 네트워크 적용안

중남미 바이어 요청 시 **China Plus One** 대안으로 한국·일본·인도·중동 제조 소스를 제시하고, Lumi AI·데이터 에이전트·감독 위원회와 연동하기 위한 소스 목록·연동 시나리오·수익 모델·검수 룰을 정리합니다.

**관련 문서**: [B2B_AGENTIC_PLAN.md](./B2B_AGENTIC_PLAN.md), [B2B_LEADS_AND_MESSAGING.md](./B2B_LEADS_AND_MESSAGING.md).

| 항목 | 내용 |

| **목적** | 중남미 바이어 요청 시 중국 외 대안(China Plus One)으로 한국·일본·인도·중동 제조 소스를 제시하고, Lumi AI·데이터 에이전트·감독 위원회와 연동. |

## 2. 국가별 제조·B2B 소스 (무료/저렴 API·사이트)

### 한국 (South Korea) — 정밀 부품·고품질 제조

| 소스 | 용도·특징 |
|------|-----------|
| **BuyKorea (KOTRA)** | 한국 공식 B2B 플랫폼. 신뢰도 높음, 무료 API로 품목별 제조사 정보. 가전·자동차 부품·IT 기기 강점. |
| **GobizKorea** (중소벤처기업진흥공단) | 중소 제조사 해외 판로 개척. 카탈로그·견적 요청 활발. |
| **Kompass Korea** | 전 세계 기업 디렉토리, 한국 제조사 데이터 정교 분류. |

### 일본 (Japan) — 정밀 기계·자동차 부품·고부가가치 소재

| 소스 | 용도·특징 |
|------|-----------|
| **J-GoodTech** | 정부 우수 중소기업 매칭. 정밀 부품·의료기기·첨단 소재. 영문 지원으로 에이전트 수집 용이. |
| **JETRO** | KOTRA 유사. TTPP DB로 수출 희망 기업 리스트. |
| **IPROS (아이프로스)** | 일본 최대 제조·기술 DB. 공장 자동화·센서·공구 등 중남미 제조 공장 부품 소스. |
| **Alibaba Japan** | 내수 위주 제조사 해외 판로. 품질·납기 중시 바이어 제안용. |

### 인도 (India) — 원자재·기계 부품·화학

| 소스 | 용도·특징 |
|------|-----------|
| **IndiaMART** | 인도 최대 B2B 마켓플레이스. 가격 경쟁력·Low MOQ 협의 용이. |
| **TradeIndia** | 중소 제조사 밀집. 농기계 부품·섬유 소싱 유리. |
| **ExportersIndia** | 수출 전문 기업 리스트. |

### 중동 (Middle East) — 에너지·건설 자재·중계 무역

| 소스 | 용도·특징 |
|------|-----------|
| **Tradekey (Middle East)** | SA·UAE 수입업자·제조사 리스트. |
| **Dubai Trade** | 두바이 재수출(Re-export) 업체. 중남미 물량 허브. |
| **Yellow Pages UAE** | 제조 공장 연락처 확보용. |

## 3. 중남미 맞춤 부품(Spare Parts) 데이터 소스

중남미(특히 DO·MX)는 노후 기계 유지보수·애프터마켓 부품 수요가 큼.

| 소스 | 용도 |
|------|------|
| **RockAuto** (US) | 자동차·산업 기기 부품. 구조화된 데이터로 가격 비교·스크래핑 용이. |
| **Parts.com** | 전 세계 OEM 부품 번호 대조 DB. |
| **Panjiva / ImportGenius** (유료) | 세관 신고 기반으로 중남미 업체의 부품 수입국·품목 파악. **역제안**("A업체가 중국에서 비싸게 사는 부품을 한국 B업체에서 더 저렴하게") 활용. |

## 4. 시스템 연동 시나리오 (에이전트 플로우)

| 단계 | 담당 | 동작 |
|------|------|------|
| **발굴** | Data Agent | BuyKorea API 등으로 한국 태양광 인버터 부품 제조사 수집. |
| **분석** | Lumi AI | DO 세관 등 데이터로 현재 수입 단가 대비 15% 절감 가능성 분석. |
| **검증** | AI 감독 위원회 | 법률: 수출 면허. 비즈니스: 인도 대비 품질. 윤리: ISO 14001 등 친환경 인증. |
| **제안** | Sales Agent | DO 에너지 업체 N곳에 "한국산 고품질 부품 직구 솔루션" WhatsApp 자동 발송. |

## 5. 글로벌 공급망 전략 (제안서 티어)

바이어 요청에 따라 Lumi AI가 선택하는 **전략적 제안** 문구.

| 티어 | 국가 | 메시지 예시 |
|------|------|-------------|
| **High-End** | 일본 | "고장 없는 정밀도가 필요하시다면 일본 J-GoodTech 인증 업체의 부품을 권장합니다." |
| **Best Value** | 한국 | "성능과 가격의 균형을 원하시면 BuyKorea 검증 한국산 제품을 제안합니다." |
| **Cost-Effective** | 인도 | "대량 소모품·일반 부품은 IndiaMART로 단가를 약 30% 낮출 수 있습니다." |

## 6. AI 감독 위원회 다국적 검수 가이드

| 감독관 | 검수 포인트 |
|--------|-------------|
| **법률** | 일본 JIS, 한국 KC/KS, 인도 ISI 등 **인증 여부** 대조 → 바이어 수입 규정 위반 방지. |
| **비즈니스** | **물류 기간**(한국/일본 30–45일, 인도 40일+ 등) 반영해 **예상 수령일** 안내. |
| **윤리** | 인도 제조사는 **아동 노동 금지·환경 규정** 준수 여부 평판 2차 검증. |

## 8. 기술 연동 방향

| 대상 | 방향 |
|------|------|
| **한국 BuyKorea** | KOTRA API로 품목별 제조사 리스트 추출 → DO 제안용 태양광·산업용 부품 필터링. (API 키·엔드포인트는 KOTRA/BuyKorea 문서 참고.) |
| **인도 IndiaMART** | 애프터마켓 부품(자동차·농기계) **단가 조회** 스크립트로 실시간 단가·마진 계산. (이용 약관·robots.txt 준수.) |
| **일본 J-GoodTech** | 도미니카 바이어 선호 자동차/기계 부품 업체 리스트 추출 시 J-GoodTech·JETRO 데이터 우선 활용. |

## 9. 적용 순서 제안

1. **소스 목록 확정**: §2·§3 표를 기준으로 한국·일본·인도·중동·부품 소스 URL·API 문서 링크를 한 시트/문서로 정리.
2. **데이터 에이전트**: BuyKorea(또는 우선 1개국) API 연동 → 제조사 리스트 수집·정규화 파이프라인 구축.
3. **감독 룰 반영**: §6 다국적 검수 가이드를 `b2bPartnerVerificationService` 또는 별도 "공급자 검증" 체크리스트에 반영(인증·물류 기간·윤리 항목).
4. **제안 티어**: §5 문구를 `b2bMessageService` 제안서 템플릿 또는 프롬프트 변수로 추가.
5. **수익 모델**: 5~12% 수수료·검수 비용·SLA 구독을 [B2B_PAYMENT_AND_COMPLIANCE_GUIDE.md](./B2B_PAYMENT_AND_COMPLIANCE_GUIDE.md)와 맞춰 정리.
